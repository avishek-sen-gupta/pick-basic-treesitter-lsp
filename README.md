# tree-sitter-pickbasic

A [tree-sitter](https://tree-sitter.github.io/) grammar for Pick BASIC, based on the 1985 PICK BASIC Reference Manual (88A00778A02).

This entire project was vibe-coded with [Claude Code](https://claude.ai/claude-code).
